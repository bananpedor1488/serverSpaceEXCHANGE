## 🔍 Тестовые сценарии

### Сценарий 1: Первая установка PIN
1. Открыть настройки приватности
2. Нажать "PIN-код"
3. Ввести 4 цифры
4. Подтвердить 4 цифры
5. Вернуться назад
6. Проверить что статус "Включен"

### Сценарий 2: Автоблокировка по таймеру
1. Установить PIN
2. Установить автоблокировку на 1 минуту
3. Оставить приложение открытым
4. Подождать 1 минуту
5. Проверить что появился экран разблокировки

### Сценарий 3: Ручная блокировка
1. Установить PIN
2. Открыть список чатов
3. Нажать кнопку замочка слева от заголовка
4. Проверить что появился экран разблокировки

### Сценарий 4: Блокировка при уходе в фон
1. Установить PIN
2. Нажать Home (свернуть приложение)
3. Открыть приложение снова
4. Проверить что появился экран разблокировки

### Сценарий 5: Биометрия
1. Установить PIN
2. Включить Face ID / Touch ID
3. Заблокировать приложение
4. Проверить что автоматически запустилась биометрия
5. Отменить биометрию
6. Ввести PIN вручную

### Сценарий 6: Серверные настройки
1. Открыть настройки приватности
2. Изменить "Кто может писать мне" на "Контакты"
3. Закрыть приложение
4. Открыть снова
5. Проверить что настройка сохранилась

### Сценарий 7: Блокировка пользователя
1. Открыть чат с пользователем
2. Открыть профиль пользователя
3. Нажать "Заблокировать"
4. Вернуться в настройки приватности
5. Открыть "Заблокированные"
6. Проверить что пользователь в списке
7. Разблокировать пользователя
