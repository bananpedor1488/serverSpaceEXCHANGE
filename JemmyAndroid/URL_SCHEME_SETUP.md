# URL Scheme Setup для Jemmy

## Что сделано:

✅ Красивая HTML страница при открытии ссылки в браузере
✅ Кнопка "Открыть в Jemmy" 
✅ Автоматический редирект в приложение
✅ Обработка custom URL scheme `jemmy://invite/{token}`

## Настройка в Xcode:

### 1. Добавить URL Scheme

1. Открой проект в Xcode
2. Выбери target **Jemmy**
3. Перейди на вкладку **Info**
4. Найди секцию **URL Types**
5. Нажми **+** (добавить)
6. Заполни:
   - **Identifier**: `com.banan.Weeky`
   - **URL Schemes**: `jemmy`
   - **Role**: Editor

### 2. Перезапусти приложение

Собери и установи на iPhone.

## Как это работает:

1. Пользователь создаёт ссылку: `https://weeky-six.vercel.app/api/u/abc123`
2. Другой пользователь открывает ссылку в Safari
3. Показывается красивая страница с:
   - Аватаром пользователя
   - Именем
   - Био
   - Кнопкой "💬 Открыть в Jemmy"
4. Автоматически пытается открыть `jemmy://invite/abc123`
5. iOS спрашивает: "Открыть в Jemmy?"
6. Приложение открывается и показывает профиль
7. Пользователь нажимает "Начать чат"

## Тестирование:

### В Safari на iPhone:
1. Создай ссылку в приложении
2. Открой в Safari
3. Должна показаться красивая страница
4. Нажми кнопку → откроется приложение

### Прямой URL:
```
jemmy://invite/abc123
```
Вставь в Safari → откроется приложение
