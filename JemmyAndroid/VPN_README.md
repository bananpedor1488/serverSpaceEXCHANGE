# Jemmy VPN - Базовая реализация

## Что сделано

Создана базовая реализация VPN сервиса для Android с поддержкой VLESS протокола.

### Основные компоненты:

1. **JemmyVpnService.kt** - VPN сервис
   - Создает TUN интерфейс
   - Обрабатывает сетевые пакеты
   - Поддерживает VLESS URL

2. **VpnScreen.kt** - UI для управления VPN
   - Список конфигураций
   - Подключение/отключение
   - Статус соединения

3. **AddVpnConfigScreen.kt** - Добавление конфигов
   - Ввод VLESS URL
   - Сохранение конфигураций

4. **VpnConfigManager.kt** - Управление конфигами
   - Сохранение в SharedPreferences
   - Выбор активного конфига

## Текущее состояние

Базовая реализация создана БЕЗ внешних зависимостей. Сервис:
- ✅ Создает VPN интерфейс
- ✅ Парсит VLESS URL
- ✅ Показывает уведомления
- ⚠️ Обработка пакетов упрощена (требует доработки)

## Что нужно для полноценной работы

Для полноценного VPN с VLESS нужно:

### Вариант 1: Использовать готовую библиотеку
Добавить в `app/build.gradle.kts`:
```kotlin
implementation("com.github.v2fly:v2ray-core:v5.12.1")
```

Или скачать готовый `.aar` файл от v2rayNG/v2fly и добавить локально.

### Вариант 2: Собрать свою библиотеку
1. Клонировать https://github.com/v2fly/v2ray-core
2. Собрать Go библиотеку для Android
3. Добавить в проект как JNI

### Вариант 3: Использовать прокси
Вместо полноценного VPN можно:
1. Запустить локальный SOCKS5 прокси
2. Использовать Android VPN API для перенаправления трафика
3. Проксировать через VLESS сервер

## Как собрать

```bash
cd serverSpaceEXCHANGE/JemmyAndroid
./gradlew assembleDebug
```

APK будет в: `app/build/outputs/apk/debug/app-debug.apk`

## Формат VLESS URL

```
vless://UUID@SERVER:PORT?encryption=none&security=tls&sni=DOMAIN&type=ws&path=/PATH&host=HOST
```

Пример:
```
vless://12345678-1234-1234-1234-123456789012@example.com:443?encryption=none&security=tls&type=ws&path=/ws
```
